INSTALLATION INSTRUCTIONS
===========================

STEP 1: Upload the Plugin
--------------------------
Option A - Via WordPress Admin:
1. Zip the entire "wordpress-export-plugin" folder
2. Go to WordPress Admin → Plugins → Add New
3. Click "Upload Plugin"
4. Choose the zip file and click "Install Now"
5. Click "Activate Plugin"

Option B - Via FTP:
1. Upload the entire "wordpress-export-plugin" folder to:
   /wp-content/plugins/
2. Go to WordPress Admin → Plugins
3. Find "Yoast Blog & SEO Exporter" and click "Activate"


STEP 2: Use the Plugin
-----------------------
1. Go to: Tools → Export Blog & SEO
2. Choose your export format (JSON recommended)
3. Select your options
4. Click "Export Blog Posts & SEO Data"
5. File will download automatically


STEP 3: Share the Export File
------------------------------
After export, share the downloaded file (.json or .csv) so we can:
- Analyze the data structure
- Create the import tool for your new website
- Map all SEO data to the new database


SAFETY GUARANTEE
================
✅ This plugin is 100% safe
✅ It only READS data - never modifies anything
✅ No database changes
✅ No file modifications
✅ Can be deleted anytime without impact

If you have any concerns, you can:
- Review the code (it's well-commented)
- Test on a staging site first
- Deactivate immediately after export
